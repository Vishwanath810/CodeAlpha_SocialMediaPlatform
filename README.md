# 🚀 CodeAlpha Social Media Platform

A full-stack social media web application built using Node.js, Express.js, MongoDB, EJS, HTML, CSS, and JavaScript.

## 📌 Features

* User Authentication (Register/Login/Logout)
* Create Posts
* Like Posts
* Comment System
* Delete Posts & Comments
* User Profiles
* Follow Users
* Responsive Professional UI
* MongoDB Database Integration

---

## 🛠 Tech Stack

### Frontend

## ⚡ Installation

### 1 Clone Repository

```bash
git clone https://github.com/Vishwanath810/CodeAlpha_SocialMediaPlatform.git
```

### 2 Install Dependencies

```bash
npm install
```

### 3 Create .env File

```env
MONGO_URI=your_mongodb_connection_string
```

### 4 Run Project

```bash
npm run dev
```
